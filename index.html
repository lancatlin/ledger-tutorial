Why Ledger

市面上的記帳軟體

  • 專有軟體，不能自己擴展
  • 多種資產，無法全部一起管理
  • 沒有隱私
  • 界面複雜，不便操作
  • 記帳太繁瑣，無法堅持

Ledger CLI

ledger-cli.org

  • 自由軟體
  • 支援任意資產：
    • 股票、Crypto、健身器材（？）
  • 純文字記帳，開 VIM 就能寫
  • 指令操作，簡單易用
  • 簡單格式，容易擴展

為什麼不用圖型介面？

  • 小、速度快（1.3MB）
  • 簡化操作
  • 客製化
  • 容易擴展、串接

複式記帳

Double-Entry Bookkeeping

複式簿記

不只是「收入」和「支出」

記錄「金錢的流動」

任一筆交易，都是兩個帳戶之間的轉移

全部的淨值加總必須為 0

表現更複雜的資產流動

ex: 買賣股票

不能單純的歸類為收入/支出

進而影響投資心態

帳戶慣例（如 GNU Cash）

Assets - Liabilities + Income - Expenses = Equity

Ledger

Income + Expenses + Assets + Liabilities + Equity = 0

黃字代表錢流出到其他帳戶，為負號

Ledger 完全不會更動你的帳本

簡單的格式，用任何的文字編輯器都可以記帳

記帳就像寫日記一樣簡單

買衣服


2022/06/30 * 買衣服
    Expenses:Clothes    $800
    Assets:Checking     -$800

發薪水


2022/06/30 * 打工
    Assets:Savings      $2000
    Income:Salary       -$2000

可空一項讓 ledger 自動平衡
